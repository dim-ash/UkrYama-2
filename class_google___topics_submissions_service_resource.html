Google_TopicsSubmissionsServiceResource::listTopicsSubmissions (   $seriesId,
  $topicId,
  $optParams = array() 
)

Searches the submissions for the specified topic within the specified series and returns the search results. (submissions.list)

Аргументи
string$seriesIdThe decimal ID of the Series.
string$topicIdThe decimal ID of the Topic within the Series.
array$optParamsOptional parameters.

string max-results Maximum number of results to return. bool includeVotes Specifies whether to include the current user's vote string start-index Index of the first result to be retrieved. string author Restricts the results to submissions by a specific author. string sort Sort order. string q Search query. bool hasAttachedVideo Specifies whether to restrict to submissions that have videos attached.
