Generate an AdSense report based on the report request sent in the query parameters. Returns the result as JSON; to retrieve output in CSV format specify "alt=csv" as a query parameter. (reports.generate)

Аргументи
string$accountIdHosted account upon which to report.
string$startDateStart of the date range to report on in "YYYY-MM-DD" format, inclusive.
string$endDateEnd of the date range to report on in "YYYY-MM-DD" format, inclusive.
array$optParamsOptional parameters.

string dimension Dimensions to base the report on. string filter Filters to be run on the report. string locale Optional locale to use for translating report output to a local language. Defaults to "en_US" if not specified. string maxResults The maximum number of rows of report data to return. string metric Numeric columns to include in the report. string sort The name of a dimension or metric to sort the resulting report on, optionally prefixed with "+" to sort ascending or "-" to sort descending. If no prefix is specified, the column is sorted ascending. string startIndex Index of the first row of report data to return.
