55 <h3>Homes?</h3>
56 <p>Each group can have a home. The home is the page where that user is redirected after login.<br/>
57 Users can have a home too. If a user has a declared home he will be redirected to this one instead theat the one of his group.<br/>
58 As you can see in the drop down list the controller will still displayed with the label you assigned to it in <b>$_permissionControl</b></p>
