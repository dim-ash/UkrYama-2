Import more rows into a table. (table.importRows)

Аргументи
string$tableIdThe table into which new rows are being imported.
array$optParamsOptional parameters.

string delimiter The delimiter used to separate cell values. This can only consist of a single character. Default is ','. string encoding The encoding of the content. Default is UTF-8. Use 'auto-detect' if you are unsure of the encoding. int endLine The index of the last line from which to start importing, exclusive. Thus, the number of imported lines is endLine - startLine. If this parameter is not provided, the file will be imported until the last line of the file. If endLine is negative, then the imported content will exclude the last endLine lines. That is, if endline is negative, no line will be imported whose index is greater than N + endLine where N is the number of lines in the file, and the number of imported lines will be N + endLine - startLine. bool isStrict Whether the CSV must have the same number of values for each row. If false, rows with fewer values will be padded with empty values. Default is true. int startLine The index of the first line from which to start importing, inclusive. Default is 0.
