1 Yii-PDF Extension
2 =================
3 
4 Небольшое расширение для Yii Framework, которое делает "обертку" для нескольких PHP-библиотек
5 (mPDF и HTML2PDF на данный момент) для конвертации HTML в PDF
6 
7 ### Ссылки
8 
9 * [Yii Framework](http://yiiframework.com/)
10 * [mPDF](http://www.mpdf1.com/) - это PHP класс для создания PDF файлов из HTML с поддержкой Unicode/UTF-8 и CJK
11 * [HTML2PDF](http://html2pdf.fr/en/default) - это PHP класс, который использует FPDF для PHP4, и TCPDF для PHP5.
12 Библиотека может конвертировать валидный HTML и XHTML в PDF
13 
14 ### Системные требования
15 
16 * Yii 1.1.9 или выше
17 * [mPDF](http://www.mpdf1.com/mpdf/download) версия 5.3 (была выпущена 2011-07-21) или выше
18 * [HTML2PDF](http://sourceforge.net/projects/phphtml2pdf/) версия 4.03 (была выпущена 2011-05-27) или выше

25 ### Установка
26 
27 * Скачайте и распакуйте расширение в директорию `protected/extensions/yii-pdf`
28 * Скачайте и распакуйте библиотеку ([mPDF](http://www.mpdf1.com/mpdf/download) and/or [HTML2PDF](http://sourceforge.net/projects/phphtml2pdf/))
29 в свою папку каталога `protected/vendors` или укажите новое значение для параметра `'librarySourcePath'` в массиве `'params'`
30 * Массив `'defaultParams'` - это массив параметров по умолчанию для конструктора выбранной библиотеки.
31 Если хотите изменить параметры по умолчанию - можете установить их в конфигурационном файле (как показано ниже).
32 При изменении - **вы должны сохранить порядок элементов массива!**
33 * В вашем конфигурационным файле `protected/config/main.php`, добавьте такие строчки:
34 
35 ```php
36 <?php
37 //...
38  'components'=>array(
39  //...
40  'ePdf' => array(
41  'class' => 'ext.yii-pdf.EYiiPdf',
42  'params' => array(
43  'mpdf' => array(
44  'librarySourcePath' => 'application.vendors.mpdf.*',
45  'constants' => array(
46  '_MPDF_TEMP_PATH' => Yii::getPathOfAlias('application.runtime'),
47  ),
48  'class'=>'mpdf', // Для некоторых "регистрочувствительных" систем
49  /*'defaultParams' => array( // Детальней: http://mpdf1.com/manual/index.php?tid=184
50  'mode' => '', // This parameter specifies the mode of the new document.
51  'format' => 'A4', // format A4, A5, ...
52  'default_font_size' => 0, // Sets the default document font size in points (pt)
53  'default_font' => '', // Sets the default font-family for the new document.
54  'mgl' => 15, // margin_left. Sets the page margins for the new document.
55  'mgr' => 15, // margin_right
56  'mgt' => 16, // margin_top
57  'mgb' => 16, // margin_bottom
58  'mgh' => 9, // margin_header
59  'mgf' => 9, // margin_footer
60  'orientation' => 'P', // landscape or portrait orientation
61  )*/
62  ),
63  'HTML2PDF' => array(
64  'librarySourcePath' => 'application.vendors.html2pdf.*',
65  'classFile' => 'html2pdf.class.php', // For adding to Yii::$classMap
66  /*'defaultParams' => array( // Детальней: http://wiki.spipu.net/doku.php?id=html2pdf:en:v4:accueil
67  'orientation' => 'P', // landscape or portrait orientation
68  'format' => 'A4', // format A4, A5, ...
69  'language' => 'en', // language: fr, en, it ...
70  'unicode' => true, // TRUE means clustering the input text IS unicode (default = true)
71  'encoding' => 'UTF-8', // charset encoding; Default is UTF-8
72  'marges' => array(5, 5, 5, 8), // margins by default, in order (left, top, right, bottom)
73  )*/
74  )
75  ),
76  ),
77  //...
78  )
79 //...
80 ```

82 ### Использование
83 
84 ```php
85 <?php
86 ...
87  public function actionIndex()
88  {
89  # mPDF
90  $mPDF1 = Yii::app()->ePdf->mpdf();
91 
92  # Вы можете с легкостью переопределить параметры по умолчанию для конструктора
93  $mPDF1 = Yii::app()->ePdf->mpdf('', 'A5');
94 
95  # render (полная страница page)
96  $mPDF1->WriteHTML($this->render('index', array(), true));
97 
98  # Загрузить таблицу стилей в документ
99  $stylesheet = file_get_contents(Yii::getPathOfAlias('webroot.css') . '/main.css');
100  $mPDF1->WriteHTML($stylesheet, 1);
101 
102  # renderPartial (только представление текущего контроллера)
103  $mPDF1->WriteHTML($this->renderPartial('index', array(), true));
104 
105  # Отрисовка картинки в документе
106  $mPDF1->WriteHTML(CHtml::image(Yii::getPathOfAlias('webroot.css') . '/bg.gif' ));
107 
108  # Вывод готового PDF
109  $mPDF1->Output();
110 
111  ////////////////////////////////////////////////////////////////////////////////////
112 
113  # у HTML2PDF очень схож синтаксис
114  $html2pdf = Yii::app()->ePdf->HTML2PDF();
115  $html2pdf->WriteHTML($this->renderPartial('index', array(), true));
116  $html2pdf->Output();
117 
118  ////////////////////////////////////////////////////////////////////////////////////
119 
120  # Пример с HTML2PDF wiki: Отправка PDF по почте
121  $content_PDF = $html2pdf->Output('', EYiiPdf::OUTPUT_TO_STRING);
122  require_once(dirname(__FILE__).'/pjmail/pjmail.class.php');
123  $mail = new PJmail();
124  $mail->setAllFrom('webmaster@my_site.net', "My personal site");
125  $mail->addrecipient('mail_user@my_site.net');
126  $mail->addsubject("Example sending PDF");
127  $mail->text = "This is an example of sending a PDF file";
128  $mail->addbinattachement("my_document.pdf", $content_PDF);
129  $res = $mail->sendmail();
130  }
131 ...
132 ```
