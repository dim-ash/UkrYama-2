Google_UsersServiceResource::listUsers (   $optParams = array())

Retrieve either deleted users or all users in a domain (paginated) (users.list)

Аргументи
array$optParamsOptional parameters.

string customer Immutable id of the Google Apps account. In case of multi-domain, to fetch all users for a customer, fill this field instead of domain. string domain Name of the domain. Fill this field to get users from only this domain. To return all users in a multi-domain fill customer field instead. int maxResults Maximum number of results to return. Default is 100. Max allowed is 500 string orderBy Column to use for sorting results string pageToken Token to specify next page in the list string query Query string for prefix matching searches. Should be of the form "key:value*" where key can be "email", "givenName" or "familyName". The asterisk is required, for example: "givenName:Ann*" is a valid query. string showDeleted If set to true retrieves the list of deleted users. Default is false string sortOrder Whether to return results in ascending or descending order.
