28 <h3>So, does userGroups find my cron job using magic?</h3>
29 <p>Of course it does! Just kidding.<br/>
30 To make userGroups aware of your cron job you'll need to put it in his own file, wherever you want to, and make sure that he is in a directory that
31 is already imported by Yii.<br/>
32 You can see what directories are imported in Yii on default and set your own in the application config file:
33 </p>
34 
35 <div class="code"><code>
36  'import'=&gt;array(
37  'application.models.*',
38  'application.components.*',
39  <span class="highlight">'ext.myFolder.*',</span>
40  ),
41 
42 </code></div>
43 
44 <p>For more informations about how to use the dot notation when importing files, read the official Yii documentation.<br/>
45 Now there's just one more thing that you need to do: tell userGroups about your cron job.<br/>
46 In the config file set a crons parameter and assign an array to it. Each value of this array must be the class name of one of yours
47 cron jobs:</p>
48 
49 <div class="code"><code>
50  'userGroups'=&gt;array(
51  'accessCode'=&gt;'ciao',
52  <span class="highlight">'crons'=&gt;array('myCronClassName'),</span>
53  ),
54 
55 </code></div>

58 If you get tired of your own cron you'll just need to delete it's name from the userGroups configuration settings inside the application
