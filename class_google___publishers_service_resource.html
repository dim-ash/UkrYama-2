Retrieves data about a single advertiser if that the requesting advertiser/publisher has access to it. Only advertisers can look up publishers. Publishers can request information about themselves by omitting the publisherId query parameter. (publishers.get)

◆ listPublishers()

Google_PublishersServiceResource::listPublishers (   $role,
  $roleId,
  $optParams = array() 
)

Retrieves data about all publishers that the requesting advertiser/publisher has access to. (publishers.list)

Аргументи
string$roleThe role of the requester. Valid values: 'advertisers' or 'publishers'.
string$roleIdThe ID of the requesting advertiser or publisher.
array$optParamsOptional parameters.

string maxResults Max number of items to return in this page. Optional. Defaults to 20. double minNinetyDayEpc Filters out all publishers that have a ninety day EPC average lower than the given value (inclusive). Min value: 0.0. Optional. int minPayoutRank A value between 1 and 4, where 1 represents the quartile of publishers with the lowest ranks and 4 represents the quartile of publishers with the highest ranks. Filters out all publishers with a lower rank than the given quartile. For example if a 2 was given only publishers with a payout rank of 25 or higher would be included. Optional. double minSevenDayEpc Filters out all publishers that have a seven day EPC average lower than the given value (inclusive). Min value 0.0. Optional. string pageToken The value of 'nextPageToken' from the previous page. Optional. string publisherCategory Caret(^) delimted list of publisher categories. Valid categories: (unclassified|community_and_content|shopping_and_promotion|loyalty_and_rewards|network|search_specialist|comparison_shopping|email). Filters out all publishers not in one of the given advertiser categories. Optional. string relationshipStatus Filters out all publishers for which do not have the given relationship status with the requesting publisher.
